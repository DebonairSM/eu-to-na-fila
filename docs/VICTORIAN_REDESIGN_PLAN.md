# Victorian Minimalistic Redesign Plan

## Design Philosophy
Combine minimalistic modern design principles with Victorian era elegance and refinement. Clean, spacious layouts with rich, deep colors and elegant typography.

## Color Palette

### Primary Colors
- **Deep Burgundy**: `#8B2635` - Primary accent, borders, highlights
- **Antique Gold**: `#C9A961` - Secondary accent, decorative elements
- **Cream/Off-White**: `#F5F1E8` - Background, text on dark
- **Charcoal Black**: `#1A1A1A` - Primary background
- **Navy Blue**: `#2C3E50` - Secondary background, depth

### Accent Colors
- **Rich Green**: `#4A5D23` - Success states (serving)
- **Deep Red**: `#7A1F1F` - Remove/delete actions
- **Warm Gray**: `#6B6B6B` - Subtle text, borders

## Typography

### Font Selection
- **Primary Font**: Elegant serif (Playfair Display, Cormorant, or similar)
  - For titles and headings
  - Large, refined, classic
- **Secondary Font**: Clean sans-serif (Inter, Roboto, or similar)
  - For body text and UI elements
  - Modern, readable

### Typography Scale
- **Title**: 3.5rem - 4rem, serif, uppercase, letter-spacing 4-6px
- **Customer Names**: 1.5rem - 2rem, serif, regular weight
- **Position Numbers**: 2rem - 2.5rem, serif, bold
- **Body Text**: 1rem, sans-serif, regular
- **Labels**: 0.875rem, sans-serif, uppercase, letter-spacing 2px

## Layout & Proportions

### Spacing
- Generous whitespace (golden ratio based)
- Card padding: 32px - 48px
- Gap between cards: 24px - 32px
- Section margins: 64px - 96px

### Card Design
- Minimal borders (1-2px)
- Subtle shadows (single, soft)
- Rounded corners: 8px - 12px (subtle)
- Clean backgrounds with slight texture
- High contrast between elements

## Icons & Visual Elements

### Icon Style
- Line icons (thin, elegant)
- Ornamental dividers (subtle flourishes)
- Classic symbols (crown, scroll, quill-inspired)
- Minimal decorative borders

### Decorative Elements
- Thin horizontal dividers with subtle patterns
- Ornamental corner brackets (very subtle)
- Classic border treatments
- Elegant separators

## Components Redesign

### Title/Header
- Large serif font, uppercase
- Subtle underline with decorative element
- Centered, generous spacing
- Deep burgundy or antique gold

### Queue Cards
- Clean white/cream background on dark
- Thin burgundy border (1-2px)
- Elegant serif for customer names
- Large, refined position badges
- Minimal shadows
- Generous padding

### Position Badges
- Circular or rounded square
- Antique gold background
- Deep burgundy or black text
- Elegant serif numbers
- Subtle border
- Classic proportions

### Buttons
- Minimal design
- Thin borders
- Elegant typography
- Subtle hover states
- Classic proportions

### Background
- Deep charcoal or navy
- Subtle texture (paper-like)
- Minimal patterns
- High contrast with cards

## Implementation Strategy

1. **Update Color Variables** - Replace current palette with Victorian colors
2. **Import Serif Font** - Add elegant serif font family
3. **Redesign Typography** - Apply new font hierarchy
4. **Simplify Cards** - Remove excessive effects, add elegance
5. **Update Icons** - Use more classic, refined icons
6. **Refine Spacing** - Apply golden ratio proportions
7. **Add Subtle Decorations** - Minimal Victorian flourishes
8. **Polish Details** - Fine-tune every element

## Key Principles
- **Less is More**: Remove unnecessary elements
- **Elegance over Flash**: Subtle, refined effects
- **Classic Proportions**: Golden ratio, balanced spacing
- **High Contrast**: Clear hierarchy, readable
- **Refined Details**: Every element carefully considered
